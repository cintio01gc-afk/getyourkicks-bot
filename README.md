# GetYourKicks Bot

Bot che legge i messaggi dal canale Discord personale e li pubblica su:
- Canale #deals del server Soleback
- Canale Telegram @getyourkickss

## Variabili d'ambiente (da impostare su Railway)

| Variabile | Valore |
|-----------|--------|
| DISCORD_BOT_TOKEN | Token del bot Discord |
| TELEGRAM_BOT_TOKEN | Token del bot Telegram |
| TELEGRAM_CHANNEL | @getyourkickss |
| WEBHOOK_SOLEBACK | URL webhook canale #deals Soleback |
| SOURCE_CHANNEL_ID | ID canale Discord da monitorare |
